TO DO LIST

https://ella-kim-bee.github.io/vue-todo-app/

CONCEPT

최근 front-end에서 많이 사용되는 framework인 vue.js를 사용하여 to do app을 만들었습니다.
To Do, Doing, Done 세 가지로 분류하여 localStorage에 저장하고 삭제하는 기능을 구현하였습니다.

FUNCTIONAL

TodoHeader, TodoSelectbox, TodoInput, MyList, TodoFooter의 components를 제작하였고,
MyList component를 To Do, Doing, Done 세 가지로 활용하였습니다.
storage에는 JSON형식으로 변환하고 일정시간 뒤에 값을 보내도록 설정해놓았고,
이를 App.vue에서 받아 created에서 동기처리를 하였습니다.

USAGE

To Do, Doing, Done 중 상태를 선택하고, 내용을 작성 후 Enter를 치거나, 우측 +버튼을 클릭하면
해당하는 상태에 내용이 추가되고 localStorage에 배열로 저장됩니다.
새로고침하면 localStorage에 저장된 내용이 일정시간 뒤엔 출력됩니다.
각 내용 우측에 휴지통 버튼을 클리하면 해당 내용만 삭제되고, 가장 하단에 있는 Clear All을 클릭하면 모든 내용이 삭제됩니다

PLANNING

HTML, CSS, javascript를 넘어서 최신의 기술력을 습득하고자 하였습니다.
component 단위로 구상하였고, 데이터를 저장하고 사용하는데 중점을 두었습니다.
